Hogwarts: The Journey of a Wizard
1. General Presentation
Project Title: Hogwarts: The Journey of a Wizard

Brief Description:

This project is a Python text-based role-playing game inspired by the saga Harry Potter.
The player follows the journey of a young wizard from their arrival in the magical world to their first major challenge at Hogwarts.

The game is divided into several chapters(four)covering:

Character creation and Diagon Alley,

The journey to Hogwarts and house sorting,

Magic classes and quizzes,

A final chapter featuring a Quidditch match simulation.

The project focuses on Python use which includes functions, conditions, loops, dictionaries, lists, randomness, and JSON file handling.
