1. General Presentation
Project Title

Hogwarts: The Journey of a Wizard

Brief Description

This project is a Python text-based role-playing game inspired by the Harry Potter universe.
The player follows the journey of a young wizard from their arrival in the magical world to their first challenges at Hogwarts.

The game is divided into four chapters:

character creation and Diagon Alley,

the journey to Hogwarts and house sorting,

magic classes and a quiz,

a final chapter with a Quidditch match.

The project focuses on basic Python programming concepts such as functions, conditions, loops, lists, dictionaries, randomness, and JSON file handling.

Development Environment

Python version: Python 3.12 or higher

External libraries: None

Only standard Python libraries have been used:
random, json, math

Usage

To run the game, navigate to the project directory and execute:

python main.py

Key Features

Character creation with attributes (courage, intelligence, loyalty, ambition)

Sorting Hat system to assign a Hogwarts house

Spell learning system using JSON files

Inventory and money management

House points system

Turn-based Quidditch match simulation

3. Control, Testing, and Validation
Input and Error Management

Text inputs are checked to avoid empty values

Numeric inputs are validated and limited

Menu choices are restricted to valid options

Purchases are blocked if the player has insufficient money

The game ends correctly if mandatory actions are not completed

Testing Strategies

The game was tested through full playthroughs:

Chapter 1: character creation and purchases

Chapter 2: dialogue choices and house assignment

Chapter 3: spell learning and quiz scoring

Chapter 4: Quidditch match progression and end conditions

Known Bugs:

Random elements can lead to different results

No major bugs were identified.
